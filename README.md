# Bantay 72 Super Admin

This is the Super Admin Dashboard for the Bantay 72 Emergency System.

## Features
- **Dashboard & Emergency Reports**: Real-time incident tracking.
- **Analytics & Reports**: Visual data representation.
- **Role-Based Access Control**: Different permissions for Super Admin and Admin.
- **User Management & Announcements**: Platform administration.

## Tech Stack
- HTML, Tailwind CSS, JavaScript
- Firebase Auth & Firestore

## Deployment
This project is configured to be deployed seamlessly on [Vercel](https://vercel.com).
